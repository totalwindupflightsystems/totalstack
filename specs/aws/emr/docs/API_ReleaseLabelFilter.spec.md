# ReleaseLabelFilter
<a name="API_ReleaseLabelFilter"></a>

The release label filters by application or version prefix.

## Contents
<a name="API_ReleaseLabelFilter_Contents"></a>

 ** Application **   <a name="EMR-Type-ReleaseLabelFilter-Application"></a>
Optional release label application filter. For example, `spark@2.1.0`.  
Type: String  
Required: No

 ** Prefix **   <a name="EMR-Type-ReleaseLabelFilter-Prefix"></a>
Optional release label version prefix filter. For example, `emr-5`.  
Type: String  
Required: No
