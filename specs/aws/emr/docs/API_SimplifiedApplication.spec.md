# SimplifiedApplication
<a name="API_SimplifiedApplication"></a>

## Contents
<a name="API_SimplifiedApplication_Contents"></a>

 ** Name **   <a name="EMR-Type-SimplifiedApplication-Name"></a>
The returned release label application name. For example, `hadoop`.  
Type: String  
Required: No

 ** Version **   <a name="EMR-Type-SimplifiedApplication-Version"></a>

Type: String  
Required: No
