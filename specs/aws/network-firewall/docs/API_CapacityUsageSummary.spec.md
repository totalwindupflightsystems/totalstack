# CapacityUsageSummary
<a name="API_CapacityUsageSummary"></a>

The capacity usage summary of the resources used by the [ReferenceSets](API_ReferenceSets.md) in a firewall.

## Contents
<a name="API_CapacityUsageSummary_Contents"></a>

 ** CIDRs **   <a name="networkfirewall-Type-CapacityUsageSummary-CIDRs"></a>
Describes the capacity usage of the CIDR blocks used by the IP set references in a firewall.  
Type: [CIDRSummary](API_CIDRSummary.md) object  
Required: No
