# LoggingConfiguration
<a name="API_LoggingConfiguration"></a>

Defines how AWS Network Firewall performs logging for a [Firewall](API_Firewall.md). 

## Contents
<a name="API_LoggingConfiguration_Contents"></a>

 ** LogDestinationConfigs **   <a name="networkfirewall-Type-LoggingConfiguration-LogDestinationConfigs"></a>
Defines the logging destinations for the logs for a firewall. Network Firewall generates logs for stateful rule groups.   
Type: Array of [LogDestinationConfig](API_LogDestinationConfig.md) objects  
Required: Yes
