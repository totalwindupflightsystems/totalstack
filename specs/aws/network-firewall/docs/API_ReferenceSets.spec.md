## Contents
<a name="API_ReferenceSets_Contents"></a>

 ** IPSetReferences **   <a name="networkfirewall-Type-ReferenceSets-IPSetReferences"></a>
The list of IP set references.  
Type: String to [IPSetReference](API_IPSetReference.md) object map  
Key Length Constraints: Minimum length of 1. Maximum length of 32.  
Key Pattern: `^[A-Za-z][A-Za-z0-9_]*$`   
Required: No
