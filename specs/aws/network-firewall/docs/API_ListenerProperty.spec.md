# ListenerProperty
<a name="API_ListenerProperty"></a>

Open port for taking HTTP or HTTPS traffic. 

## Contents
<a name="API_ListenerProperty_Contents"></a>

 ** Port **   <a name="networkfirewall-Type-ListenerProperty-Port"></a>
Port for processing traffic.  
Type: Integer  
Required: No

 ** Type **   <a name="networkfirewall-Type-ListenerProperty-Type"></a>
Selection of HTTP or HTTPS traffic.  
Type: String  
Valid Values: `HTTP | HTTPS`   
Required: No
