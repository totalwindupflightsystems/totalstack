# DeleteVolumeOpenZFSConfiguration
<a name="API_DeleteVolumeOpenZFSConfiguration"></a>

A value that specifies whether to delete all child volumes and snapshots. 

## Contents
<a name="API_DeleteVolumeOpenZFSConfiguration_Contents"></a>

 ** Options **   <a name="FSx-Type-DeleteVolumeOpenZFSConfiguration-Options"></a>
To delete the volume's child volumes, snapshots, and clones, use the string `DELETE_CHILD_VOLUMES_AND_SNAPSHOTS`.  
Type: Array of strings  
Array Members: Maximum number of 1 item.  
Valid Values: `DELETE_CHILD_VOLUMES_AND_SNAPSHOTS`   
Required: No
