# OpenZFSNfsExport
<a name="API_OpenZFSNfsExport"></a>

The Network File System (NFS) configurations for mounting an Amazon FSx for OpenZFS file system. 

## Contents
<a name="API_OpenZFSNfsExport_Contents"></a>

 ** ClientConfigurations **   <a name="FSx-Type-OpenZFSNfsExport-ClientConfigurations"></a>
A list of configuration objects that contain the client and options for mounting the OpenZFS file system.   
Type: Array of [OpenZFSClientConfiguration](API_OpenZFSClientConfiguration.md) objects  
Array Members: Maximum number of 25 items.  
Required: Yes
