# ActionConfiguration
<a name="API_ActionConfiguration"></a>

Represents information about an action configuration.

## Contents
<a name="API_ActionConfiguration_Contents"></a>

 ** configuration **   <a name="CodePipeline-Type-ActionConfiguration-configuration"></a>
The configuration data for the action.  
Type: String to string map  
Key Length Constraints: Minimum length of 1. Maximum length of 50.  
Value Length Constraints: Minimum length of 1. Maximum length of 1000.  
Required: No
