# FargateProfileHealth
<a name="API_FargateProfileHealth"></a>

The health status of the Fargate profile. If there are issues with your Fargate profile's health, they are listed here.

## Contents
<a name="API_FargateProfileHealth_Contents"></a>

 ** issues **   <a name="AmazonEKS-Type-FargateProfileHealth-issues"></a>
Any issues that are associated with the Fargate profile.  
Type: Array of [FargateProfileIssue](API_FargateProfileIssue.md) objects  
Required: No
