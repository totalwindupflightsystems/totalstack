# ElasticsearchDataSourceConfig
<a name="API_ElasticsearchDataSourceConfig"></a>

Describes an OpenSearch data source configuration.

As of September 2021, Amazon Elasticsearch service is Amazon OpenSearch Service. This configuration is deprecated. For new data sources, use [OpenSearchServiceDataSourceConfig](API_OpenSearchServiceDataSourceConfig.md) to specify an OpenSearch data source.
