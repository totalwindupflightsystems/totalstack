# EventBridgeDataSourceConfig
<a name="API_EventBridgeDataSourceConfig"></a>

Describes an Amazon EventBridge bus data source configuration.

## Contents
<a name="API_EventBridgeDataSourceConfig_Contents"></a>

 ** eventBusArn **   <a name="appsync-Type-EventBridgeDataSourceConfig-eventBusArn"></a>
The ARN of the event bus. For more information about event buses, see [Amazon EventBridge event buses](https://docs.aws.amazon.com/eventbridge/latest/userguide/eb-event-bus.html).  
Type: String  
Required: Yes
