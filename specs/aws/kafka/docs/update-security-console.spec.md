# Update Amazon MSK cluster security settings using the AWS Management Console
<a name="update-security-console"></a>

1. Sign in to the AWS Management Console, and open the Amazon MSK console at [https://console.aws.amazon.com/msk/home?region=us-east-1\#/home/](https://console.aws.amazon.com/msk/home?region=us-east-1#/home/).

1. Choose the MSK cluster that you want to update.

1. In the **Security settings** section, choose **Edit**.

1. Choose the authentication and encryption settings that you want for the cluster, then choose **Save changes**.
