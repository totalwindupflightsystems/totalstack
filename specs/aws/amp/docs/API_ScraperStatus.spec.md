# ScraperStatus
<a name="API_ScraperStatus"></a>

The `ScraperStatus` structure contains status information about the scraper.

## Contents
<a name="API_ScraperStatus_Contents"></a>

 ** statusCode **   <a name="prometheus-Type-ScraperStatus-statusCode"></a>
The current status of the scraper.  
Type: String  
Valid Values: `CREATING | UPDATING | ACTIVE | DELETING | CREATION_FAILED | UPDATE_FAILED | DELETION_FAILED`   
Required: Yes
