# LoggingFilter
<a name="API_LoggingFilter"></a>

Filtering criteria that determine which queries are logged.

## Contents
<a name="API_LoggingFilter_Contents"></a>

 ** qspThreshold **   <a name="prometheus-Type-LoggingFilter-qspThreshold"></a>
The Query Samples Processed (QSP) threshold above which queries will be logged. Queries processing more samples than this threshold will be captured in logs.  
Type: Long  
Valid Range: Minimum value of 0.  
Required: Yes
