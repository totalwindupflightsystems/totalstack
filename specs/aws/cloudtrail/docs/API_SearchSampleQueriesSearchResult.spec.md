# SearchSampleQueriesSearchResult
<a name="API_SearchSampleQueriesSearchResult"></a>

 A search result returned by the `SearchSampleQueries` operation.

<a name="API_SearchSampleQueriesSearchResult_Contents"></a>

 ** Description **   <a name="awscloudtrail-Type-SearchSampleQueriesSearchResult-Description"></a>

Type: String  
Required: No

 ** Name **   <a name="awscloudtrail-Type-SearchSampleQueriesSearchResult-Name"></a>

Type: String  
Required: No
