# ProtectionGroupArbitraryPatternLimits
<a name="API_ProtectionGroupArbitraryPatternLimits"></a>

 ** MaxMembers **   <a name="AWSShield-Type-ProtectionGroupArbitraryPatternLimits-MaxMembers"></a>
