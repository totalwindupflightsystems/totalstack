# ValidationExceptionField
<a name="API_ValidationExceptionField"></a>

Provides information about a particular parameter passed inside a request that resulted in an exception.

## Contents
<a name="API_ValidationExceptionField_Contents"></a>

 ** message **   <a name="AWSShield-Type-ValidationExceptionField-message"></a>
The message describing why the parameter failed validation.  
Type: String  
Required: Yes

 ** name **   <a name="AWSShield-Type-ValidationExceptionField-name"></a>
The name of the parameter that failed validation.  
Type: String  
Required: Yes
