# Limit
<a name="API_Limit"></a>

Specifies how many protections of a given type you can create.

## Contents
<a name="API_Limit_Contents"></a>

 ** Max **   <a name="AWSShield-Type-Limit-Max"></a>
The maximum number of protections that can be created for the specified `Type`.  
Type: Long  
Required: No

 ** Type **   <a name="AWSShield-Type-Limit-Type"></a>
The type of protection.  
Type: String  
Required: No
