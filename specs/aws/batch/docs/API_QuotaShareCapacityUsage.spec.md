# QuotaShareCapacityUsage
<a name="API_QuotaShareCapacityUsage"></a>

The capacity usage for a quota share, including units of compute capacity and quantity of resources being used.

## Contents
<a name="API_QuotaShareCapacityUsage_Contents"></a>

 ** capacityUnit **   <a name="Batch-Type-QuotaShareCapacityUsage-capacityUnit"></a>
The unit of compute capacity for the capacity usage.  
Type: String  
Required: No

 ** quantity **   <a name="Batch-Type-QuotaShareCapacityUsage-quantity"></a>
The quantity of capacity being used.  
Type: Double  
Required: No
