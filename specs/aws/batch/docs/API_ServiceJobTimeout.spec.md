# ServiceJobTimeout
<a name="API_ServiceJobTimeout"></a>

The timeout configuration for service jobs. 

## Contents
<a name="API_ServiceJobTimeout_Contents"></a>

 ** attemptDurationSeconds **   <a name="Batch-Type-ServiceJobTimeout-attemptDurationSeconds"></a>
The maximum duration in seconds that a service job attempt can run. After this time is reached, AWS Batch terminates the service job attempt.  
Type: Integer  
Required: No
