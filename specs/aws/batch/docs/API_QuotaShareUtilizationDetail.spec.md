# QuotaShareUtilizationDetail
<a name="API_QuotaShareUtilizationDetail"></a>

An object that represents the capacity utilization details of all quota shares associated with a single job queue.

## Contents
<a name="API_QuotaShareUtilizationDetail_Contents"></a>

 ** topCapacityUtilization **   <a name="Batch-Type-QuotaShareUtilizationDetail-topCapacityUtilization"></a>
A list of the top capacity utilizations across quota shares associated with a job queue.  
Type: Array of [QuotaShareCapacityUtilization](API_QuotaShareCapacityUtilization.md) objects  
Required: No
