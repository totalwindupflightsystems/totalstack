# Mathematical operators
<a name="mathematical-operators"></a>

Timestream for LiveAnalytics supports the following mathematical operators.


| Operator | Description | 
| --- | --- | 
| \+ | Addition | 
| - | Subtraction | 
| \* | Multiplication | 
| / | Division (integer division performs truncation) | 
| % | Modulus (remainder) |
