# TimestampRange
<a name="API_TimestampRange"></a>

Specifies a time range for filtering certificates.

## Contents
<a name="API_TimestampRange_Contents"></a>

**Note**  
In the following list, the required parameters are described first.

 ** End **   <a name="ACM-Type-TimestampRange-End"></a>
The end of the time range. This value is inclusive.  
Type: Timestamp  
Required: No

 ** Start **   <a name="ACM-Type-TimestampRange-Start"></a>
The start of the time range. This value is inclusive.  
Type: Timestamp  
Required: No
