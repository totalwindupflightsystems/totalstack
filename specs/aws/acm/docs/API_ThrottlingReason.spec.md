# ThrottlingReason
<a name="API_ThrottlingReason"></a>

A description of why a request was throttled.

## Contents
<a name="API_ThrottlingReason_Contents"></a>

**Note**  
In the following list, the required parameters are described first.

 ** reason **   <a name="ACM-Type-ThrottlingReason-reason"></a>
A description of why a request was throttled.  
Type: String  
Required: No

 ** resource **   <a name="ACM-Type-ThrottlingReason-resource"></a>
The resource that causes the request to be throttled.  
Type: String  
Required: No
