# MedicalScribePatientContext

## Contents
<a name="API_streaming_MedicalScribePatientContext_Contents"></a>

 ** Pronouns **   <a name="transcribe-Type-streaming_MedicalScribePatientContext-Pronouns"></a>
The patient's preferred pronouns that the user wants to provide as a context for clinical note generation .  
Type: String  
Valid Values: `HE_HIM | SHE_HER | THEY_THEM`   
Required: No
