## Amazon Kendra
<a name="Welcome_Amazon_Kendra"></a>

This is the *Amazon Kendra API Reference*. Amazon Kendra is an intelligent search service that uses natural language processing and advanced machine learning algorithms to return specific answers to questions using your data.

For an overview of the Amazon Kendra APIs, see [Amazon Kendra Actions](https://docs.aws.amazon.com/kendra/latest/APIReference/API_Operations_Amazon_Kendra.html).

For information about the IAM access control permissions you need to use this API, see [IAM roles for Amazon Kendra](https://docs.aws.amazon.com/kendra/latest/dg/iam-roles.html) in the *Amazon Kendra Developer Guide*.

The following resources provide additional information:
+  * [Setting up Amazon Kendra](https://docs.aws.amazon.com/kendra/latest/dg/setup.html) * 
+  * [Amazon Kendra CLI Reference](https://awscli.amazonaws.com/v2/documentation/api/latest/reference/kendra/index.html) *

## Amazon Kendra Intelligent Ranking
<a name="Welcome_Amazon_Kendra_Intelligent_Ranking"></a>

Amazon Kendra Intelligent Ranking uses Amazon Kendra semantic search capabilities to intelligently re-rank a search service's results.

For an overview of the Amazon Kendra Intelligent Ranking APIs, see [Amazon Kendra Intelligent Ranking Actions](https://docs.aws.amazon.com/kendra/latest/APIReference/API_Operations_Amazon_Kendra_Intelligent_Ranking.html).
