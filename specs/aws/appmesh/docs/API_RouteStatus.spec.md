# RouteStatus
<a name="API_RouteStatus"></a>

An object that represents the current status of a route.

## Contents
<a name="API_RouteStatus_Contents"></a>

 ** status **   <a name="appmesh-Type-RouteStatus-status"></a>
The current status for the route.  
Type: String  
Valid Values: `ACTIVE | INACTIVE | DELETED`   
Required: Yes
