# MeshServiceDiscovery
<a name="API_MeshServiceDiscovery"></a>

An object that represents the service discovery information for a service mesh.

## Contents
<a name="API_MeshServiceDiscovery_Contents"></a>

 ** ipPreference **   <a name="appmesh-Type-MeshServiceDiscovery-ipPreference"></a>
The IP version to use to control traffic within the mesh.  
Type: String  
Valid Values: `IPv6_PREFERRED | IPv4_PREFERRED | IPv4_ONLY | IPv6_ONLY`   
Required: No
