# VirtualServiceStatus
<a name="API_VirtualServiceStatus"></a>

 ** status **   <a name="appmesh-Type-VirtualServiceStatus-status"></a>
The current status of the virtual service.  
Type: String  
Valid Values: `ACTIVE | INACTIVE | DELETED`   
Required: Yes
