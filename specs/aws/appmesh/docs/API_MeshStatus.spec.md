# MeshStatus
<a name="API_MeshStatus"></a>

An object that represents the status of a service mesh.

## Contents
<a name="API_MeshStatus_Contents"></a>

 ** status **   <a name="appmesh-Type-MeshStatus-status"></a>
The current mesh status.  
Type: String  
Valid Values: `ACTIVE | INACTIVE | DELETED`   
Required: No
