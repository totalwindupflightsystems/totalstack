# VirtualRouterSpec
<a name="API_VirtualRouterSpec"></a>

An object that represents the specification of a virtual router.

## Contents
<a name="API_VirtualRouterSpec_Contents"></a>

 ** listeners **   <a name="appmesh-Type-VirtualRouterSpec-listeners"></a>
The listeners that the virtual router is expected to receive inbound traffic from.  
Type: Array of [VirtualRouterListener](API_VirtualRouterListener.md) objects  
Required: No
