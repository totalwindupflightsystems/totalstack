# GatewayRouteHostnameMatch
<a name="API_GatewayRouteHostnameMatch"></a>

An object representing the gateway route host name to match.

## Contents
<a name="API_GatewayRouteHostnameMatch_Contents"></a>

 ** exact **   <a name="appmesh-Type-GatewayRouteHostnameMatch-exact"></a>
The exact host name to match on.  
Type: String  
Length Constraints: Minimum length of 1. Maximum length of 253.  
Required: No

 ** suffix **   <a name="appmesh-Type-GatewayRouteHostnameMatch-suffix"></a>
The specified ending characters of the host name to match on.  
Type: String  
Length Constraints: Minimum length of 1. Maximum length of 253.  
Required: No
